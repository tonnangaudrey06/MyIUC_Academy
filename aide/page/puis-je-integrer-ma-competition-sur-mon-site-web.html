Puis-je intégrer ma compétition sur mon site web ?

Gérer une compétition

Centre d'aide

Oui. Pour cela, il suffit de vous rendre dans le Tableau de bord de votre compétition où vous trouverez la section "Code à intégrer sur votre site". Vous pouvez personnaliser la largeur et la hauteur de la boîte à afficher sur votre site (les paramètres proposés devrait cependant convenir à la plupart des sites). Cliquez sur "Ok" pour générer le code : il vous suffit ensuite de le copier et de le coller à l'endroit de votre site où vous voulez afficher votre compétition.

Si votre site utilise un système de gestion de contenus comme Wordpress ou Footeo, veillez à coller le code en mode "HTML" ou "code source".
